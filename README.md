# Generative-AI-for-Fraud-Detection-and-Compliance
Functional prototype that demonstrate how to leverage GenAI in real-world application

# Objective
This project apply the concepts developing a generative AI-powered fraud detection system:
-  create synthetic data
-  train a model
-   integrate it into a simple compliance workflow.

# Step 1: Generate Synthetic Data
Tool: GenAI tool

Task: Generate a synthetic dataset that mimics real-world financial transactions. The dataset should include fields like transaction mamount, location, time, and a fraud flag.

Output: Save this dataset as a CSV file, which will be used for training
your AI model.

# Step 2: Train a Fraud Detection Model
Tool: Python with Scikit learn

Task: Use the synthetic dataset to train a generative AI model that can predict whether a transaction is fraudulent. 

Output: A trained model that can classify transactions as either fraudulent or legitimate.
